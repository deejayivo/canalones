# Tejados y Canalones Hidalgo

Web estática sencilla para publicar servicios, historia, trabajos y datos de contacto de Tejados y Canalones Hidalgo.

## Pendiente antes de publicar

- Confirmar que el teléfono `699 707 343` es el definitivo para llamadas y WhatsApp.
- Añadir email si quieren recibir presupuestos por correo.
- Confirmar el dominio definitivo y sustituir `https://www.tejadosycanaloneshidalgo.es/` en `index.html`, `robots.txt` y `sitemap.xml`.
- Añadir fotos reales de trabajos en la sección de galería.
- Añadir el enlace real de Google Business Profile en el campo `sameAs` del JSON-LD.
- Si tienen dirección física pública, añadirla al bloque `address`.

## SEO local recomendado

- Crear o reclamar la ficha de Google Business Profile.
- Usar exactamente el mismo nombre, teléfono y zona de servicio en web, Google y redes.
- Subir fotos reales de trabajos con nombres descriptivos.
- Pedir reseñas a clientes tras cada trabajo.
- Conectar el dominio con Google Search Console y enviar `sitemap.xml`.
